## 📚 Prompts 解析
项目的 Prompts 编写包含两层：
- global_prompts.md：全局提示，包含了对占卜流程的描述，和限制条件。
- prompt_xxx.md：针对不同角色的提示词，包含了角色设定和占卜分析的提示词。

在组装最终的提示词时，合并规则为 `global_prompts + prompt_xxx + 知识库信息`。

其中，知识库信息是根据用户抽取的牌阵，从内置的塔罗牌知识库中提取的相关信息。此部分内容经XML格式化处理。

需要注意的是，部分角色的人设可能和占卜分析的提示词存在冲突，**如果想要最准确的占卜分析结果，建议使用默认角色prompt_default.md**（比如雌小鬼人设一般会想尽办法把好的结果往坏处解释）。
